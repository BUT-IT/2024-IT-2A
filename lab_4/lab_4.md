# **Задание**

## **Часть 1. Built-in Objects и Built-in Functions**

### 1. Работа с встроенными объектами

Выполнить следующие действия в отдельном файле `lab.js`:

1. Создать переменную `text = "Hello World"`.

   * Используя методы встроенного объекта `String`, выполнить:

     * перевод строки в верхний регистр
     * поиск подстроки `"World"`
     * заменить `"World"` на `"JS"`

2. Создать массив чисел `[5, 12, 8, 130, 44]`.

   * Используя методы встроенного объекта `Array`:

     * найти все элементы больше 10
     * отсортировать массив по возрастанию
     * удалить последний элемент (pop)

3. Работа с `Math`:

   * Сгенерировать случайное число от 1 до 100
   * Найти квадратный корень числа 144
   * Округлить число `4.67` вниз и вверх

4. Работа с `Date`:

   * Получить текущую дату
   * Вывести год, месяц и день по отдельности
   * Создать дату своего дня рождения

## **Часть 3. Class-based Inheritance**

Создать файл `classes.js` и выполнить:

1. Создать класс `Animal`:

   * свойства: `name`, `age`
   * метод `info()` — выводит информацию о животном

2. Создать класс `Dog`, который наследует `Animal`:

   * добавить свойство `breed`
   * добавить метод `bark()` — выводит звук собаки
   * переопределить метод `info()` (использовать `super.info()`)

3. Создать класс `Cat`, наследующий `Animal`:

   * добавить свойство `color`
   * добавить метод `meow()`

4. Создать объекты классов `Dog` и `Cat`, вызвать их методы.
